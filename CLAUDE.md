# Project Rules

This repository is for robotic tail research on quadruped robots.

## Main goals

1. Literature search about mechanism-based robots.
2. Robotic tail design for Unitree Go2.
3. IsaacLab simulation for tail-assisted stability.
4. MuJoCo simplified model for landing and balance.
5. Stability metrics, parameter sweep, and report writing.

## Safety rules

- Do not use sudo.
- Do not modify system files.
- Do not access private network addresses.
- Do not delete files unless explicitly asked.
- Do not read files outside this repository unless explicitly asked.

## Workflow rules

- For literature search, create Markdown files under `docs/`.
- Do not modify URDF, MuJoCo, or IsaacLab files during literature search tasks.
- Do not fabricate citations.
- If a paper, DOI, code, or video link is uncertain, mark it as `needs verification`.
- Prefer tables with clear columns.
- After editing, summarize changed files.
